# Runwall App

A modern web application built with React, TypeScript, and Vite.

---

## 📌 Project Overview

Runwall App is a scalable frontend application structured for
maintainability and performance.\
It uses modern tooling and best practices including TypeScript (strict
mode), modular structure, and optimized build configuration.

---

## 🛠 Tech Stack

- React
- TypeScript (Strict Mode)
- Vite
- Tailwind CSS (if applicable)
- ESLint / Prettier (recommended)

---

## 🚀 Getting Started

### 1️⃣ Install dependencies

    npm install

### 2️⃣ Start development server

    npm run dev

App will run on:

    http://localhost:5173

---

## 🏗 Build for Production

    npm run build

Preview production build:

    npm run preview

---

## ⚙️ Environment Configuration

If environment variables are used, create a `.env` file in the root
directory:

    VITE_API_BASE_URL=http://localhost:3000
